# AFM.ai Dashboard — AI Operating System (Phase 1)

## Live now

| Area | Path | Capability |
|------|------|------------|
| Command Center | `/dashboard` | Pipeline diagram + quick links |
| Swarm AI | `/dashboard/swarm` | Single, Swarm, Auto, Debate, Research |
| Adaptive Workspace | `/dashboard/workspace` | UI adapts to code/video/marketing |
| AI Router | `/dashboard/core/router` | Intent analyzer + task routing table |
| Workflows | `/dashboard/automation/workflows` | Viral video, campaign, auto content |
| Memory | `/dashboard/memory/*` | Browser-local memory (Phase 1) |
| Integrations | `/dashboard/integrations` | AI Connect grid (Phase 2 hooks) |
| Personality | `/dashboard/settings` | 6 personalities + 3 operating modes |

## APIs

- `POST /api/afm/swarm` — multi-mode orchestration
- `POST /api/afm/intent` — intent + workspace + subtask routes
- `POST /api/afm/workflow` — run workflow planning + swarm summary
